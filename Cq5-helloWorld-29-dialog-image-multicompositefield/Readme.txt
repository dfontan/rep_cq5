CQ5 AND MULTICOMPOSITEFIELD


Multicompositefield is element of dialog which can contain a lot of other elements. You can add and 
remove these elements by "+" and "-".
	Multicompositefield is extended multifield and it enables you to add and remove images and videos.
	

To work with multicompositefield you have to:

1. Add folder "widgets"
You have to add this folder to your application folder (for instance "webapp"). It contains all javascripts
which handles multicompositefield.

2. Use xtype "multicompositefield"
You have to use this xtype in your dialog. You have to use all javascripts from this dialog.
