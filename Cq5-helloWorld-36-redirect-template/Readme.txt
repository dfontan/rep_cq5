CQ5 AND REDIRECT TEMPLATE

Main goal of this project is to show how to create redirect template. It means how to create page 
which redirect you to other page. It is very useful in CQ because here you create create hierarchy using
pages but sometimes you don`t want to display some pages. They exist only for this hierarchy purpose. 

In author mode you can define path to page where you want to be redirected.



USAGE

This project contains an example content. Link:
http://localhost:4502/cf#/content/webapp.html
or
http://localhost:4503/content/webapp.html

You will be redirected to /content/webapp/page.html
