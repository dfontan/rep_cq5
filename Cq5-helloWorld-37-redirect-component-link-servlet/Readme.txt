CQ5 AND REDIRECT LINK CONNECTED WITH SERVLET


This tutorial is about creating redirect link component. User can add this component to content 
and define link:
* text: displayed text on link;
* url: where user should be redirected after choosing link.

A servlet with selector "REDIRECT" is used for this redirection.
