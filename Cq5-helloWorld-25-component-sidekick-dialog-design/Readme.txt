CQ5 AND DESIGN DIALOG


Design dialog is a dialog which is displayed when you are in desing mode
and when you click "Edit". It enables to set some properties which are
common for all components of specific type. For instance you can set
width and hight for component with type image and then all components
of this type with have this width and hight. So using this dialog you
can set desing of some component.

In order to user desing dialog you have to:
- add desing_dialog.xml with properties
- you can get properties in jsp by ${currentStyle['propertyName']}
